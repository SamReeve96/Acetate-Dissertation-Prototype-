Welcome to Acetate!

Acetate is a tool that lets you annotate web pages.

If Acetate is open you can annotate this very page!

Just right click on an element or highlighted text and select "Annotate" in chrome's context menu

This will create a card, click annotate to save the card

If an element is annotated it will be highlighted and glow a colour (default is orange)

Cards automatically get out of the way of content, want to see them? Can see them a few different ways, Either hover over a card on the right of the window, hover over an annotated element or press the "o" (I promise I'll make this an actual shortcut soon.)

You can even annotate some elements, try annotating this image to say how lovely it is
